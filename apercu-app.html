Fonctionnalités clés

🧠

Coach IA contextuel

Messages personnalisés par semaine, heure du jour et performance récente. Le coach adapte ses conseils à ton RPE moyen et tes jours de repos.

🎯

Mode Séance immersif

Interface plein écran : timer Nexus Pulse, saisie poids/reps/RPE, suggestions de progression, et estimation 1RM automatique.

📊

Calcul auto des charges

Toutes les charges de 40% à 102% auto-calculées à partir de tes 1RM. Arrondi au 2.5 kg. Modifie un 1RM → tout se met à jour.

🔄

Gestion de cycles

Archive automatique de chaque cycle (1RM, séances, PRs). Nouveau cycle avec suggestion de 1RM basée sur tes performances.

✏️

Programme personnalisable

Override d'exercices, séries/reps et pourcentages directement dans l'app. Ton programme s'adapte à toi.

📱

PWA installable

Hors-ligne, installable sur l'écran d'accueil. Service Worker, Wake Lock pour garder l'écran allumé en séance.

Mode Séance — Détails

Nexus Pulse Timer

Timer circulaire animé avec vibration. Durées ajustables (1:30, 2:00, 3:00, 5:00).

Saisie rapide

Poids pré-rempli + boutons ±2.5 kg. Sélection RPE visuelle (6 à 10).

Estimation 1RM auto

Si RPE ≥ 8 et poids > 1RM actuel → suggestion via formule d'Epley.

Mode Companion

Vue simplifiée plein écran. Idéal pour les lunettes embuées.

Wake Lock

Écran allumé pendant toute la séance. Réactivation automatique.

Persistance séance

Séries faites et poids sauvegardés. Recharge sans perte.

État global (CanditoContext)

  • 1RM athlète (Squat / Bench / Deadlift)
  • Sessions complétées + Logs détaillés
  • Records personnels (PRs)
  • Historique des cycles
  • Overrides programme
  • Migration auto du schéma (v1→v6)

Persistance

  • IndexedDB via idb-keyval (principal)
  • localStorage fallback + migration auto
  • Session en cours persistée séparément
  • Export/Import JSON complet
  • 100% local — pas de backend

PWA & Offline

  • Service Worker avec cache-first
  • Manifest.json (standalone, portrait)
  • Banner d'installation iOS/Android
  • Mise à jour avec prompt utilisateur
  • Web Push notifications

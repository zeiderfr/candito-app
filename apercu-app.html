Fonctionnalités clés

🧠

Coach IA contextuel

Messages personnalisés par semaine, heure du jour et RPE récent. Le coach adapte ses conseils selon ta dernière séance, tes jours de repos et ta fatigue accumulée.

🎯

Mode Séance immersif

Plein écran avec Nexus Pulse Timer, saisie poids/reps/RPE par série, estimation 1RM automatique (Epley), mode Companion simplifié, et Wake Lock.

📊

Calcul auto des charges

40% à 102% auto-calculés depuis tes 1RM. Arrondi au 2.5 kg. Modifie un 1RM → toutes les charges du programme se mettent à jour instantanément.

🔄

Gestion de cycles

Archive automatique de chaque cycle (1RM, séances, PRs). Lancement d'un nouveau cycle avec suggestion de 1RM basée sur tes logs.

✏️

Programme personnalisable

Override d'exercices, séries/reps et pourcentages directement dans l'app. Ton programme s'adapte à tes besoins, pas l'inverse.

📱

PWA installable

Hors-ligne, installable sur écran d'accueil. Service Worker cache-first, Wake Lock, Web Push notifications d'entraînement.

Structure des composants

App
├── CanditoProvider          ← State global (IndexedDB + migrations v1→v6)
│   ├── ToastProvider        ← Notifications toast
│   └── AppContent
│       ├── AppLayout        ← Layout + BottomNav (5 onglets) + Swipe
│       ├── Dashboard        ← Accueil
│       │   ├── CoachCard    ← Messages contextuels par semaine/heure/RPE
│       │   ├── NextSessionHero  ← Prochain exercice + poids animé
│       │   ├── LastSessionCard  ← Résumé dernière séance + tonnage
│       │   └── AthleteStats     ← 1RM grid + progress ring animé
│       ├── Warmup           ← Mobilité + Gamme montante (bas/haut)
│       ├── Programme        ← Sélecteur semaine + SessionCards
│       │   ├── WeekSelector     ← Pills S1-S6 avec dots progression
│       │   ├── SessionCard      ← Exercices + charges calculées
│       │   └── FocusMode        ← Séance immersive plein écran
│       │       ├── NexusPulse Timer  ← Timer circulaire SVG animé
│       │       ├── Set Logger        ← Poids ±2.5 / RPE / reps
│       │       ├── 1RM Suggestion    ← Epley auto si RPE ≥ 8
│       │       └── Companion Mode    ← Vue simplifiée grand format
│       ├── Nutrition        ← Macros périodisées + Timing repas
│       └── Profil           ← Navigation stack interne
│           ├── AthleteHero      ← Avatar, nom éditable, total
│           ├── SessionTimeline  ← Barre 17 séances
│           ├── PRSection        ← Records personnels CRUD
│           ├── LiftProgressGraph ← Graphe trajectoire de force
│           ├── CycleHistory     ← Cycles archivés
│           ├── ProgramMainView  ← Personnalisation programme
│           └── Settings         ← Export/Import JSON

État global

  • 1RM athlète (SQ/BP/DL)
  • Sessions + Logs détaillés par série
  • Records personnels (PRs)
  • Historique multi-cycles
  • Overrides programme (exos/séries/%)
  • Migration auto v1→v6

Persistance

  • IndexedDB via idb-keyval
  • localStorage fallback + migration
  • Session en cours persistée séparément
  • Export/Import JSON complet
  • 100% local — zéro backend

PWA & Offline

  • Service Worker cache-first
  • Manifest standalone portrait
  • Banner install iOS/Android
  • Update prompt utilisateur
  • Web Push notifications
